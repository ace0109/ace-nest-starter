# 错误代码参考

## 系统错误 (1xxxx)

| 代码  | 描述         | HTTP状态 |
| ----- | ------------ | -------- |
| 10001 | 系统内部错误 | 500      |
| 10002 | 服务不可用   | 503      |
| 10003 | 维护中       | 503      |

## 认证错误 (2xxxx)

| 代码  | 描述     | HTTP状态 |
| ----- | -------- | -------- |
| 20001 | 未认证   | 401      |
| 20002 | 权限不足 | 403      |
| 20003 | 令牌过期 | 401      |
| 20004 | 令牌无效 | 401      |

## 用户错误 (3xxxx)

| 代码  | 描述       | HTTP状态 |
| ----- | ---------- | -------- |
| 30001 | 用户不存在 | 404      |
| 30002 | 用户已存在 | 409      |
| 30003 | 密码错误   | 400      |
| 30004 | 账户已锁定 | 403      |

## 业务错误 (4xxxx)

| 代码  | 描述       | HTTP状态 |
| ----- | ---------- | -------- |
| 40001 | 验证失败   | 400      |
| 40002 | 资源不存在 | 404      |
| 40003 | 操作失败   | 400      |
| 40004 | 重复操作   | 409      |

## 外部服务错误 (5xxxx)

| 代码  | 描述           | HTTP状态 |
| ----- | -------------- | -------- |
| 50001 | 外部服务错误   | 502      |
| 50002 | 外部服务超时   | 504      |
| 50003 | 外部服务不可用 | 503      |
